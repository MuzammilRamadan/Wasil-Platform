How It Works

One platform, three roles

Community Member

Report disease cases, find nearby mobile clinics, and access emergency health instructions for your family.

Organization

Deploy mobile clinics, manage field operations, and respond to community service requests in affected areas.

MOH System Admin

Monitor the entire epidemic response system, approve clinic requests, and oversee all registered users.

Platform Features

Everything you need for epidemic response

Real-time Epidemic Map

Track active outbreaks across regions with live severity indicators and case counts.

Clinic Requests

Organizations can request mobile clinic deployments to epidemic-hit areas instantly.

Case Reporting

Community members can report disease symptoms and cases directly from their phones.

Admin Dashboard

Full oversight for MOH admins — manage clinics, users, and response operations.

Multilingual Support

Full Arabic and English localization for all users across Sudan.

Secure Authentication

Role-based access with Supabase authentication, keeping data safe and private.
